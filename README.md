This repository contains a Vite + React frontend and a minimal NestJS backend. The frontend is configured to build into the `docs/` folder so it can be deployed to GitHub Pages.

Quick start (Windows PowerShell):

Frontend:

```powershell
cd C:\Users\Dasam\Stark2248\frontend
npm install
npm run dev
```

Backend:

```powershell
cd C:\Users\Dasam\Stark2248\backend
npm install
npm run start:dev
```

Build and deploy to GitHub Pages:

```powershell
cd C:\Users\Dasam\Stark2248\frontend
npm install
npm run deploy
```

Notes:
- The frontend build outputs to `docs/`, which GitHub Pages can serve from the repository root.
- Update the site content with your resume info in the frontend `src/components` files.
